[Configurar MySQL](#configurar-mysql)\n4. [Configurar Python e Dependências](#configurar-python-e-dependências)\n5. [Executar o Programa](#executar-o-programa)\n6. [Usar o Sistema](#usar-o-sistema)\n7. [Troubleshooting](#troubleshooting)\n\n---\n\n## 🔧 INSTALAÇÃO DO GIT\n\n### Windows\n1. Abra o navegador (Chrome, Firefox, Edge, etc.)\n2. Acesse: `https://git-scm.com/download/win`\n3. Clique em \"Click here to download manually\" (em azul)\n4. Após baixar, abra o arquivo `.exe` que foi baixado\n5. Clique em \"Next\" > \"Next\" > \"Next\" até terminar\n6. Selecione a opção \"Use Git from the Windows Command Prompt\" e clique \"Next\"\n7. Clique em \"Install\" e aguarde a instalação\n8. Clique em \"Finish\"\n\n### Linux (Ubuntu/Debian)\n1. Abra o Terminal\n2. Digite o comando:\n```bash\nsudo apt update\nsudo apt install git\n```\n3. Pressione Enter e digite sua senha quando solicitado\n\n### macOS\n1. Abra o Terminal\n2. Digite o comando:\n```bash\nbrew install git\n```\n3. Se não tiver Homebrew instalado, acesse: `https://brew.sh` e siga as instruções\n\n---\n\n## 📥 CLONAR O REPOSITÓRIO\n\n### Windows\n1. Crie uma pasta chamada \"Projetos\" no seu Desktop\n2. Clique com o botão direito dentro da pasta vazia\n3. Selecione \"Git Bash Here\" (se não aparecer, seu Git não foi instalado corretamente)\n4. Uma janela preta aparecerá\n5. Digite o comando:\n```bash\ngit clone https://github.com/kakuzu62/escala-servico.git\n```\n6. Pressione Enter\n7. Aguarde a conclusão (verá mensagens na tela)\n8. Digite:\n```bash\ncd escala-servico\n```\n9. Pressione Enter\n\n### Linux/macOS\n1. Abra o Terminal\n2. Digite:\n```bash\nmkdir ~/Projetos\ncd ~/Projetos\ngit clone https://github.com/kakuzu62/escala-servico.git\ncd escala-servico\n```\n3. Pressione Enter após cada comando\n\n---\n\n## 🗄️ CONFIGURAR MYSQL\n\n### Passo 1: Instalar MySQL\n\n#### Windows\n1. Acesse: `https://dev.mysql.com/downloads/mysql/`\n2. Procure por \"MySQL Community Server\"\n3. Clique no botão verde \"Download\"\n4. Clique em \"No thanks, just start my download\"\n5. Abra o arquivo `.msi` que foi baixado\n6. Clique em \"Next\" > \"Accept License\" > \"Next\"\n7. Selecione \"Server only\" e clique \"Next\"\n8. Clique \"Next\" > \"Next\" até chegar em \"MySQL Server 8.0\"\n9. Escolha \"Standalone MySQL Server / Classic MySQL Server\" > \"Next\"\n10. Em \"Config Type\", deixe \"Development Computer\" selecionado > \"Next\"\n11. Deixe a porta como `3306` > \"Next\"\n12. Clique \"Next\" > \"Add MySQL Server to Windows Service\" > \"Next\"\n13. Em \"MySQL Server User Configuration\", selecione \"Use Legacy Authentication Method\" > \"Next\"\n14. Digite uma senha para o usuário root (ex: 123456) > \"Next\"\n15. Clique em \"Execute\" > \"Finish\" > \"Finish\"\n\n#### Linux (Ubuntu/Debian)\n1. Abra o Terminal\n2. Digite:\n```bash\nsudo apt update\nsudo apt install mysql-server\n```\n3. Pressione Enter e digite sua senha\n4. Quando perguntado \"Do you want to continue?\", digite `y` e pressione Enter\n\n#### macOS\n1. Abra o Terminal\n2. Digite:\n```bash\nbrew install mysql\nbrew services start mysql\n```\n3. Pressione Enter\n\n### Passo 2: Acessar MySQL e Criar o Banco de Dados\n\n#### Windows\n1. Pressione `Windows + R`\n2. Digite `cmd` e pressione Enter\n3. Digite:\n```bash\nmysql -u root -p\n```\n4. Pressione Enter\n5. Digite a senha que você definiu (ex: 123456) e pressione Enter\n6. O prompt mudará para `mysql>`\n\n#### Linux/macOS\n1. Abra o Terminal\n2. Digite:\n```bash\nmysql -u root -p\n```\n3. Pressione Enter\n4. Digite a senha (se não tiver definido, só pressione Enter) e pressione Enter\n5. O prompt mudará para `mysql>`\n\n### Passo 3: Executar o Script SQL\n\n1. Você está no prompt `mysql>`\n2. Abra o arquivo `database/schema.sql` com um editor de texto (Bloco de Notas, VS Code, etc.)\n3. Copie TODO o conteúdo do arquivo\n4. Volte para o prompt do MySQL\n5. Cole o conteúdo (Ctrl + V ou Cmd + V)\n6. Pressione Enter\n7. Aguarde a execução (levará alguns segundos)\n8. Deve aparecer \"Query OK\" várias vezes\n9. Digite:\n```sql\nexit\n```\n10. Pressione Enter para sair do MySQL\n\n### Passo 4: Configurar Credenciais\n\n1. Abra a pasta do projeto com um editor (VS Code recomendado)\n2. Procure pela pasta `config`\n3. Abra o arquivo `config.ini`\n4. Procure pela seção `[DATABASE]`\n5. Altere as linhas:\n```ini\nhost = localhost\nport = 3306\nuser = root\npassword = 123456        <- ALTERE PARA SUA SENHA DO MYSQL\ndatabase = escala_servico\n```\n6. Salve o arquivo (Ctrl + S ou Cmd + S)\n\n---\n\n## 🐍 CONFIGURAR PYTHON E DEPENDÊNCIAS\n\n### Passo 1: Verificar Instalação do Python\n\n#### Windows\n1. Pressione `Windows + R`\n2. Digite `cmd` e pressione Enter\n3. Digite:\n```bash\npython --version\n```\n4. Pressione Enter\n5. Deve aparecer algo como \"Python 3.9.0\" ou superior\n6. Se aparecer \"comando não encontrado\", instale Python em: `https://www.python.org/downloads/`\n\n#### Linux/macOS\n1. Abra o Terminal\n2. Digite:\n```bash\npython3 --version\n```\n3. Pressione Enter\n4. Deve aparecer a versão do Python\n\n### Passo 2: Criar Ambiente Virtual\n\n#### Windows\n1. Abra a pasta do projeto no Command Prompt\n2. Navegue para a pasta do projeto:\n```bash\ncd C:\\Users\\SeuUsuario\\Projetos\\escala-servico\n```\n3. Digite:\n```bash\npython -m venv venv\n```\n4. Pressione Enter\n5. Aguarde alguns minutos (uma pasta `venv` será criada)\n\n#### Linux/macOS\n1. Abra o Terminal\n2. Navegue para a pasta do projeto:\n```bash\ncd ~/Projetos/escala-servico\n```\n3. Digite:\n```bash\npython3 -m venv venv\n```\n4. Pressione Enter\n5. Aguarde alguns minutos\n\n### Passo 3: Ativar Ambiente Virtual\n\n#### Windows\n1. No Command Prompt (ainda na pasta do projeto), digite:\n```bash\nvenv\\Scripts\\activate\n```\n2. Pressione Enter\n3. O prompt deve mudar para `(venv) C:\\...` (note o \"(venv)\" no início)\n\n#### Linux/macOS\n1. No Terminal (ainda na pasta do projeto), digite:\n```bash\nsource venv/bin/activate\n```\n2. Pressione Enter\n3. O prompt deve mudar para `(venv) user@machine:...` (note o \"(venv)\" no início)\n\n### Passo 4: Instalar Dependências\n\n1. Com o ambiente virtual ativado (veja o \"(venv)\" no prompt)\n2. Digite:\n```bash\npip install -r requirements.txt\n```\n3. Pressione Enter\n4. Aguarde a instalação (pode levar 5-10 minutos)\n5. Verá linhas como \"Successfully installed PyQt5 8.2.0\", etc.\n6. Quando terminar, o prompt volta ao normal\n\n---\n\n## ▶️ EXECUTAR O PROGRAMA\n\n### Windows\n1. Abra o Command Prompt na pasta do projeto\n2. Certifique-se de que o ambiente virtual está ativado (veja \"(venv)\" no prompt)\n3. Digite:\n```bash\npython main.py\n```\n4. Pressione Enter\n5. A janela do programa deve aparecer em alguns segundos\n\n### Linux/macOS\n1. Abra o Terminal na pasta do projeto\n2. Certifique-se de que o ambiente virtual está ativado (veja \"(venv)\" no prompt)\n3. Digite:\n```bash\npython main.py\n```\n4. Pressione Enter\n5. A janela do programa deve aparecer em alguns segundos\n\n---\n\n## 💻 USAR O SISTEMA\n\n### Primeira Execução - Criar Usuário Admin\n\n#### Opção 1: Via MySQL (Recomendado)\n1. Você está dentro do sistema\n\n---\n\n## 📖 GUIA DE USO DO SISTEMA\n\n### Aba: Escalas\n\n#### Criar Nova Escala\n1. Clique na aba \"Escalas\"\n2. Clique no botão \"Criar Nova Escala\"\n3. Selecione a função no dropdown \"Função\"\n4. Selecione a pessoa no dropdown \"Pessoa\"\n5. Escolha a data no calendário \"Data\"\n6. Clique em \"Criar\"\n7. Mensagem de sucesso deve aparecer\n\n#### Listar e Filtrar Escalas\n1. Clique na aba \"Escalas\"\n2. Clique em \"Listar Escalas com Filtros\"\n3. A janela de listagem abrirá\n4. Aplique filtros:\n   - **Função**: Selecione a função desejada ou deixe \"Todas\"\n   - **Pessoa**: Selecione a pessoa ou deixe \"Todas\"\n   - **Tipo**: Selecione \"Preta\", \"Vermelha\" ou \"Todas\"\n   - **Data Início**: Escolha a data inicial\n   - **Data Fim**: Escolha a data final\n5. Clique em \"Filtrar\"\n6. Os resultados aparecerão na tabela\n\n#### Editar Escala\n1. Na janela de listagem, clique em uma linha da tabela para selecioná-la\n2. Clique em \"Editar\"\n3. A janela de edição abrirá\n4. Altere os dados desejados\n5. Clique em \"Salvar\"\n6. A escala será atualizada\n\n#### Deletar Escala\n1. Na janela de listagem, clique em uma linha da tabela para selecioná-la\n2. Clique em \"Deletar\"\n3. Uma mensagem de confirmação aparecerá\n4. Clique em \"Sim\" para confirmar\n5. A escala será deletada\n\n---\n\n### Aba: Funções\n\n#### Criar Nova Função\n1. Clique na aba \"Funções\"\n2. Clique em \"Criar Nova Função\"\n3. Digite o nome da função (ex: \"Médico\", \"Enfermeiro\", \"Técnico\")\n4. Digite a descrição (opcional)\n5. Clique em \"Criar\"\n6. Mensagem de sucesso deve aparecer\n\n---\n\n### Aba: Pessoas\n\n#### Adicionar Nova Pessoa\n1. Clique na aba \"Pessoas\"\n2. Clique em \"Adicionar Pessoa\"\n3. Preencha os campos:\n   - **Nome**: Nome completo da pessoa\n   - **Email**: Email da pessoa (opcional)\n   - **Telefone**: Telefone da pessoa (opcional)\n   - **Função**: Selecione a função no dropdown\n4. Clique em \"Adicionar\"\n5. Mensagem de sucesso deve aparecer\n\n---\n\n### Aba: Relatórios\n\n#### Gerar Relatório\n1. Clique na aba \"Relatórios\"\n2. Clique em \"Gerar Relatório\"\n3. Escolha a data desejada no calendário\n4. Clique em \"Gerar\"\n5. A tabela será preenchida com as escalas do dia\n6. As colunas mostram: Função | Pessoa | Tipo Escala\n\n#### Exportar para PDF\n1. Na janela de relatório, após gerar o relatório\n2. Clique em \"Exportar PDF\"\n3. Um arquivo PDF será salvo na pasta do projeto\n4. Nome do arquivo: `Relatorio_Escala_DD_MM_YYYY.pdf`\n\n---\n\n### Menu: Dashboard (Novo)\n\n#### Acessar Dashboard\n1. Clique no menu \"Arquivo\"\n2. Clique em \"Dashboard\" (se não aparecer, será adicionado em breve)\n3. A janela do dashboard abrirá com 5 abas\n\n#### Aba: Estatísticas Gerais\n1. Mostra:\n   - Total de Funções\n   - Total de Pessoas\n   - Escalas deste mês\n   - Afastamentos ativos\n\n#### Aba: Escalas por Função\n1. Tabela com:\n   - Nome da Função\n   - Total de Escalas\n2. Ordenado do maior para o menor\n\n#### Aba: Pessoas\n1. Duas sub-abas:\n   - **Mais Escalas**: Pessoas com mais escalas atribuídas\n   - **Menos Escalas**: Pessoas com menos escalas atribuídas\n2. Mostra: Pessoa | Função | Total de Escalas\n\n#### Aba: Afastamentos\n1. Duas sub-abas:\n   - **Ativos**: Pessoas que estão em afastamento no momento\n   - **Próximos**: Afastamentos programados para os próximos 30 dias\n2. Mostra: Pessoa | Função | Tipo | Data Início | Data Fim\n\n#### Aba: Folgas\n1. Mostra a contagem de folgas de cada pessoa\n2. Colunas: Pessoa | Folgas | Ranking\n3. Selecione a função no dropdown para filtrar\n\n---\n\n## 🔐 Gerenciar Perfis de Usuários\n\n### Perfis Disponíveis\n1. **Administrador**: Acesso total ao sistema\n2. **Criador de Escala**: Pode criar e editar escalas\n3. **Visualizador**: Apenas leitura de dados\n\n### Criar Novo Usuário (Via MySQL)\n1. Reinstale Python: `https://www.python.org/downloads/`\n2. Marque a opção \"Add Python to PATH\" durante a instalação\n3. Reinicie o computador\n\n### Erro: \"MySQL não foi encontrado\"\n**Solução:**\n1. Reinstale MySQL conforme instruções acima\n2. Certifique-se que o serviço do MySQL está rodando\n\n### Erro: \"Não foi possível conectar ao banco de dados\"\n**Solução:**\n1. Verifique se o MySQL está rodando\n2. Verifique os dados em `config/config.ini`\n3. Certifique-se que executou o script SQL\n\n### Erro ao instalar dependências\n**Solução:**\n1. Atualize o pip:\n```bash\npip install --upgrade pip\n```\n2. Tente instalar novamente:\n```bash\npip install -r requirements.txt\n```\n\n### Programa abre mas nada aparece\n**Solução:**\n1. Verifique se tem um usuário cadastrado no MySQL\n2. Verifique se o MySQL está rodando\n3. Consulte a aba \"Troubleshooting\" acima\n\n---\n\n## 📞 SUPORTE\n\nSe encontrar problemas:\n1.
